# React Elasticsearch GUI

## Start in dev mode
Starting backend server and GUI in a single command
```
yarn dev
```
[Elasticsearch Simple GUI](http://localhost:3000/)

## Start server
```
yarn server
```
[Search on backend](http://localhost:9201/query?q=gr%C3%B8nn)

## Start GUI
```
yarn start
```
[Elasticsearch Simple GUI](http://localhost:3000/)

### Reference
